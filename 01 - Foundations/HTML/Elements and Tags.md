# Elements and Tags

## HTML Tags
- Tags = HTML ke building blocks
- **Opening tag** = `<p>` (angle brackets mein keyword)
- **Closing tag** = `</p>` (forward slash + keyword)
- **Element** = opening tag + content + closing tag

```html
<p>some text content</p>
```

## Element ke 3 Parts
1. Opening tag → `<p>`
2. Content → `some text content`
3. Closing tag → `</p>`

---

## Void Elements
- Kuch elements ka **closing tag nahi hota**
- Yeh content wrap nahi karte
- e.g. `<br>` (line break), `<img>` (image)
- Self closing `<br />` bhi dikh sakta hai — valid hai but discouraged

---

## Semantic HTML
- Sahi content ke liye sahi tag use karna
- 2 fayde:
    - **SEO** → search engine better samjhega
    - **Accessibility** → screen readers ke liye helpful

> [!summary] Definition
> HTML Element = Opening Tag + Content + Closing Tag
> Void Element = sirf ek tag, koi content nahi
