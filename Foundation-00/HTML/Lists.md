# Lists


## Unordered List `<ul>`
- Order matter nahi karta (shopping list, etc.)
- Bullet points se display hota hai

```html
<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
</ul>
```

## Ordered List `<ol>`
- Order matter karta hai (steps, top 10, etc.)
- Numbers se display hota hai

```html
<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Code</li>
</ol>
```

> [!summary] Quick Reference
> `<ul>` = unordered list (bullet points)
> `<ol>` = ordered list (numbers)
> `<li>` = list item (dono mein use hota hai)
>
