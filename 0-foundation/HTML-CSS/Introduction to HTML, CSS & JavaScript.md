# Introduction to HTML, CSS & JavaScript

## What is HTML?
- HTML = **HyperText Markup Language**
- Webpage ka raw data/structure
- Text, links, buttons, lists, cards — sab HTML se banta hai

## What is CSS?
- CSS = **Cascading Style Sheets**
- HTML ko style deta hai
- Color, font, position, size — sab CSS se hota hai

## What is JavaScript?
- Logic add karta hai webpage mein
- Webpage ko interactive banata hai

## HTML vs CSS vs JavaScript

HTML = Structure/Content
CSS = Style/Design
JavaScript = Logic/Interactivity

> [!important] Note
> HTML aur CSS **programming languages nahi hain** —
> yeh sirf information present karte hain.
> JavaScript ek actual programming language hai.

> [!summary] Analogy 
> HTML = ghar ki deewarein aur kamre
> CSS = paint, furniture, decoration
> JavaScript = bijli, paani, doors ka kaam karna
