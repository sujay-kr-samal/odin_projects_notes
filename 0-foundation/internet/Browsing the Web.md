# Browsing the Web

> Related: [[How the internet works]]

## Key Terms

### Web Page
- Ek document jo browser mein display hota hai
- HTML mein likha hota hai
- Embed kar sakta hai: images, videos, scripts, styles

### Website
- Web pages ka **collection** jo ek domain name share karte hain
- Pages aapas mein links se connected hote hain
- e.g. `wikipedia.org` ek website hai

### Web Server
- Ek computer jo websites **host** karta hai
- Browser request karta hai = web server response deta hai
- Ek web server **multiple websites** host kar sakta hai

### Web Service
- Software jo internet pe requests handle karta hai
- e.g. image resizer, weather API, login system

### Search Engine
- Ek web service jo doosre web pages dhundhne mein help karta hai
- e.g. Google, Bing, DuckDuckGo

> [!summary] Library Analogy 
> Library = Web Server
> Section (Science, Math) = Website
> Books = Web Pages
> Search Index = Search Engine

---

## Browser vs Search Engine

- **Browser** = software jo web pages display karta hai (Chrome, Firefox)
- **Search Engine** = service jo web pages dhundhne mein help karta hai (Google)
- Dono alag hain! Browser ek tool hai, Search Engine ek service hai

---

## How the Web Works

1. Tu URL type karta hai browser mein
2. Browser **HTTP request** bhejta hai web server ko
3. Web server **HTTP response** bhejta hai (HTML file)
4. Browser HTML parse karta hai aur requests (images, CSS, JS)
5. Sab resources load hone ke baad page display hota hai

> [!summary] Definition
> HTTP (Hypertext Transfer Protocol) = language jisme
> browser aur server baat karte hain.

## Searching Effectively

- **MDN** = web technology ke liye
- **Stack Overflow** = specific coding problems ke liye
- Language ka naam search mein zaroor dalo
    - `fibonacci sequence JavaScript`

### Advanced Search Tips
- `"exact phrase"` = exact match ke liye
- `term -exclude` = kuch exclude karne ke liye
- `term1 OR term2` = koi ek term
- `intitle:term` = page title mein term
