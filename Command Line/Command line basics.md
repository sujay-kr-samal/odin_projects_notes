# Command Line Basics

## What is CLI?
- CLI (Command Line Interface) = text based interface to interact with computer
- Developer ka sabse important tool
- Terminal, Shell, Command Line — sab same cheez hai

---

## Essential Commands

### Navigation
- `pwd` = current directory dikhao (Print Working Directory)
- `ls` = folder contents dikhao
- `cd foldername` = folder mein jao
- `cd ~` = home directory pe jao
- `cd ..` = ek step peeche jao

### Files & Folders
- `mkdir foldername` = new folder banao
- `touch filename.txt` = new file banao
- `rm filename` = file delete karo
- `rm -r foldername` = folder delete karo
- `mv oldname newname` = rename/move karo

### Pro Tips
- `Tab` = auto complete command/path
- `Ctrl+Shift+C` = copy in terminal
- `Ctrl+Shift+V` = paste in terminal
- `code .` = current folder VS Code mein open karo

---

## Opening VSCode from Terminal
```bash
code .        # current folder open karo
code file.txt # specific file open karo
```
