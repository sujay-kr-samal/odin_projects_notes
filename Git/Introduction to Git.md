# Introduction to Git

> Related: [[Setting Up Git]]

## What is Git?
- Git = **version control system** (epic save button! 💾)
- Normal save = sirf current state save hoti hai
- Git save = **differences + full history** save hoti hai
- Local machine pe kaam karta hai

## Git vs Text Editor
| Text Editor | Git |
|-------------|-----|
| Sirf ek file save | Poori history save |
| essay-draft1, draft2... | Ek file, sab versions |
| No record keeping | Complete record |

## Git vs GitHub
- **Git** = local version control tool
- **GitHub** = remote/cloud storage for Git projects
- Git = local, GitHub = remote

## Why Git?
- Individual → apna project history dekh sako
- Team → sab ka contribution track hota hai
- Portfolio → GitHub pe projects dikhta hai employers ko

> [!summary] Definition
> Git = local version control system.
> GitHub = remote hosting service for Git repositories.
